# 第二里程碑最小化设计

## 🎯 核心玩法设计

### 游戏目标
- **胜利条件**：摧毁敌方主营地
- **失败条件**：我方主营地被摧毁

### 经济循环
```
资源采集 → 建造建筑 → 生产单位 → 战斗扩张
     ↑                              ↓
     └───────── 资源再生 ──────────┘
```

### 基础建筑链
```
主营地 (初始) → 发电厂 → 采矿场 → 坦克工厂 → 坦克
```

## ⚙️ 最小化系统设计

### 资源系统（仅2种）
- **资金**：用于建造和生产
- **电力**：限制建筑数量

### 建筑列表（仅4种）
1. **主营地**
   - 核心建筑，被毁游戏结束
   - 可建造其他基础建筑

2. **发电厂**
   - 造价：500资金
   - 提供：10电力
   - 建造时间：15秒

3. **采矿场**
   - 造价：800资金
   - 位置：必须建在资源点上
   - 功能：每5秒+50资金

4. **坦克工厂**
   - 造价：1000资金
   - 消耗：5电力
   - 功能：生产坦克

### 单位列表（仅1种）
- **基础坦克**
  - 造价：300资金
  - 生产时间：20秒
  - 功能：移动、攻击

## 📱 手机操作方案

### 界面布局
```
┌─────────┬─────┐
│ 小地图  │资金 │
│        │电力 │
├─────────┼─────┤
│         │    │
│  游戏   │ 建 │
│  主视角  │ 造 │
│         │ 按 │
│         │ 钮 │
├─────┬───┼─────┤
│选择 │移动│生产 │
│按钮 │按钮│按钮 │
└─────┴───┴─────┘
```

### 核心操作
1. **建造流程**
   - 点击右下角[建造]按钮
   - 选择建筑图标
   - 拖动到有效位置（绿色高亮）
   - 点击确认建造

2. **单位选择**
   - 单选：直接点击单位
   - 多选：左下角切换选择模式后框选
   - 编队：长按选择组→分配数字键

3. **移动控制**
   - 选中单位后点击地面移动
   - 底部[攻击移动]按钮：移动中自动攻击

## 📊 平衡性设计

### 初始设置
```
每个玩家：
- 主营地 x1（提供10电力）
- 初始资金：1000
- 基础坦克 x3
- 附近资源点 x1
```

### 时间节奏
```
前期（0-2分钟）：经济建设
- 建造发电厂 → 采矿场 → 坦克工厂

中期（2-5分钟）：军事积累  
- 持续生产坦克
- 寻找扩张机会

后期（5分钟+）：决战阶段
- 大规模坦克战
- 攻击敌方主营地
```

### 资源平衡
```
发电厂：500资金 → 10电力（回本时间：无直接资金回报）
采矿场：800资金 → 每5秒+50（回本时间：80秒）
坦克工厂：1000资金 + 5电力
坦克：300资金 + 20秒生产时间
```

## 🚀 开发重点

### 核心功能优先级
1. **经济系统**：资金采集、电力管理
2. **建造系统**：建筑放置、队列管理
3. **生产系统**：坦克生产、单位控制
4. **战斗系统**：坦克移动、自动攻击

### 体验优化重点
1. **手机操作**：触屏友好、响应迅速
2. **性能稳定**：帧率保持30FPS以上
3. **网络同步**：经济数据准确同步

这个最小化设计确保第二里程碑可以在合理时间内完成，同时提供完整的RTS经济循环体验。所有元素都经过精简，只保留最核心的功能和资源。
