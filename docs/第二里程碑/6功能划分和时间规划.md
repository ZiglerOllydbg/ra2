# 功能划分

- 游戏初始化：服务器同步【1】
  - 增加初始矿石
  - 增加主城建筑、坦克
  - 增加矿源信息，位置和矿源的矿石数量
- 游戏内地图逻辑
  - 读取二进制地图
  - 设置MapManager的阻挡信息和格子信息
  - 根据地块信息，绘制显示地图
- 经济系统【2】
  - 采矿场：间隔一段时间，矿石增加。
  - 电力：建造电厂，增加10点电力，建造坦克工厂，消耗5点电力。
  - UI
    - 显示矿石总数，电力最大值（电力初始值+电厂*10）和当前使用电力（耗电建筑耗电总和）。
  - 实现：经济System，每帧计算资金和电力。
- 建筑系统【3】
  - 打开UI，显示可以建造的建筑。当前全部显示：电厂，采矿场，战车工厂。
  - 选择建筑，判断条件：
    - 建造电力、金币判定
    - 建造阻挡判定⭐
    - 【特殊判定】采矿场，判断在矿源点上。
  - 拖到地图，自动吸附。⭐
  - 确认建造，扣除消耗。
  - 建造中，进度建造，分阶段。透明建筑显示，此时也可以受到攻击，可以和建成以后一样处理。
  - 建造完成：电厂增加电力，采矿场间隔添加矿石，战车工厂可以生产坦克。
- 战斗
  - 坦克攻击顺序调整
  - 建筑支持攻击：要计算建筑尺寸，不是攻击中心点。⭐
  - 建筑血量系统
    - 建筑受损状态
    - 建筑损毁状态
    - 建筑移除
  - 胜负判定
- 操作【⭐⭐⭐】
  - 框选和编组⭐⭐
  - 单位移动
  - 相机移动⭐⭐
- 其他
  - 地图编辑器⭐⭐
    - 绘制一个简单的地图
  - 寻路RVO2替换或者改A*算法⭐⭐


## 地图阻挡
- 可行走地块        = 0 
- 不可行走地块（红色方块）      = 1 
- 草地（浅绿色方块）             = 4
- 山地 （棕色方块）            = 8
- 树源 （深绿色方块）            = 16

### 地图编辑器
- 创建地图，参数：地图尺寸大小，例如：128*128
- 添加，删除地块类型：当前只需要类型和名称、描述。例如：4、草地、可以行走
- 选择地块，鼠标拖动改变格子数据
- 保存地图，保存为一个文件，二进制文件
- 读取地图，二进制文件读取

> 以上的核心实现是为了实现地图可以分为3路，可以采矿。其他草地、山地、树源可以没有。
