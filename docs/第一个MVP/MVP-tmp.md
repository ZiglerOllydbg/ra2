# 第一个MVP🍅🍅
## 需要验证什么问题？
- 微信小游戏用户也想玩类似红警的RTS游戏，核心是操作多个单位，使用（策略）战斗。


## 包含哪些内容？
- 【重点】1v1人机对战：简单的AI即可。
- 新手引导：哪里造兵，怎么打人。没有引导，转化率为0。
- 最小化元素：主城，采矿场，电厂，坦克工厂，坦克。
- 数值：造坦克速度要快，坦克数量要多，战斗要有爽快感。30秒一场战斗，2-3分钟一场游戏
- 寻路：要顺畅
- 战斗：(受伤)显示血量
- 场景：需要阻挡和水面或者山体
- 美术资源：使用卡通3D。直接买现成的Asset Store资源包。
- 音乐资源：要有合适的音效


## 使用taptap
- 寻找硬核和高质量用户
- 通过30秒视频，吸引玩家
- 首批5-10人，观察操作困难点
- 第一批测试50-100人（taptap内测玩家）
- 发布视频，带上话题 #RTS #红警 #独立游戏。
