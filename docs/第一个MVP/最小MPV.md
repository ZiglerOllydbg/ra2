# Ra2最小MVP

## 目标
- 1v1人机对战（只要单机）
- 新手引导
- 数值：造坦克速度要快，坦克数量要多，战斗要有爽快感。30秒一场战斗，2-3分钟一场游戏
- 寻路：要顺畅
- 战斗：(受伤)显示血量
- 场景：需要阻挡和水面或者山体
- 美术资源：使用卡通3D。直接买现成的Asset Store资源包。
- 音乐资源：要有合适的音效


# 验证关键指标

## 1、玩家想在手机上玩RTS游戏？
- 目标大于40%（可以考虑具体指标）
- 关键：如果激活率<40%，说明核心假设（玩家想玩RTS）被证伪，需立即停掉或重做方向。

## 2、新手引导是否有效？
- 引导完成后，开始游戏，引导必须简单到“几乎无法失败”
- 目标大于70%
- 将引导压缩到15秒内（例如：点击“造坦克”→自动建造→点击“攻击”）
- 埋点：在引导最后加一个按钮“开始战斗”，记录完成率。

## 3、玩家是否喜欢爽快战斗？
- 单场时间2-3分钟，战斗小于30秒
- 大于3场/玩家，事件埋点



## 埋点
- 用Firebase（免费）埋点：game_start、guide_complete、battle_end、player_exit

- 标题：【RTS Mvp】2分钟战红警！手机也能玩！
